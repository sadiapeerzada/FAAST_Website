## Tech Stack

The website is built using a contemporary React-based framework, optimized for scalability, responsiveness, and visual appeal.

* **Frontend Framework:** React.js
* **Routing:** React Router DOM
* **Styling:** Tailwind CSS and shadcn/ui
* **Icons:** Lucide React
* **Animations:** Custom CSS keyframes
* **Build & Deployment:** Base44

## Design Philosophy

The FAAST website follows a **finance-meets-futurism** aesthetic, aligning with the theme of innovation and precision in FinTech.
Its visual direction is defined by:

* Deep blue and cyan gradients to evoke trust, intelligence, and technology
* Subtle neon glow effects symbolizing innovation and energy
* Structured layouts that emphasize clarity and professionalism
